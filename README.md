## 0x1
Zero X POC

### How to execute the project
```
# Terminal 1
nvm install v10.1.0 &&
nvm use v10.1.0 &&
rm -rf node_modules &&
npm i &&
npm run download_snapshot &&
npm install -g ganac he-cli &&
npm ganache

# Terminal 2
npm start
```

### Verifying accounts founds
Instal Mist and run
```
/Applications/Mist.app/Contents/MacOS/Mist --rpc http://localhost:8545
```
